Graph interactive template¶

class DCRgraph(props)¶

Responsive DCR graph display. Event execution requests are available > the public DCR smart contract is called if the node is public.

DCRgraph.DCRgraph.componentWillMount()¶

Loads the smart contract stored markings

DCRgraph.DCRgraph.componentDidMount()¶

Fits the graph display into the user window, and sets graph listeners.

DCRgraph.DCRgraph.loadContract()¶

Connects to DCR-manager smart contrat and calls the markings.

DCRgraph.DCRgraph.fetchBCid()¶

Fetches the corresponding blockchain id of the clicked event before calling the SC.

DCRgraph.DCRgraph.runBCCheck()¶

Calls the DCR-manager smart contract to check whether an event is executable. Catches the execution status, and updates all projections accordingly via a backend call.

DCRgraph.DCRgraph.setUpListeners()¶

Listens to the graph for any node click. If a click occurs on a node, an API request is sent to the backend: if the event is private, its execution is directly processed. If it is public, the function runBCCheck is called.

Component loading and displaying the blockchain public markings (in the form [event_id,markings,dataHash(opt)])

PublicMarkings.PublicMarkings.componentWillMount()¶

updates labels to display choreography names instead of ids, and connects to web3.js via connectsToBC()

PublicMarkings.PublicMarkings.connectsToBC()¶

connects to web3.js

PublicMarkings.PublicMarkings.checkHash()¶

optional function used to check the authenticity of a data set by the user (the hash is stored in the smart contract)
